# IntentText Hub

The [IntentText Hub](https://hub.dotit.uts.qa) is the template gallery for `.it` files: official templates across 8 domains, browse-and-copy from the web app.

| Domain       | Count | Examples                                          |
| ------------ | ----- | ------------------------------------------------- |
| Agent        | 8     | Workflow definitions, agent pipelines             |
| Book         | 6     | Chapters, manuscripts, long-form structure        |
| Business     | 23    | Proposal, quarterly report, meeting minutes       |
| Developer    | 5     | API spec, ADR, incident report                    |
| Editorial    | 8     | Articles, newsletters, publications               |
| Organization | 11    | Contact directories, policies, org documents      |
| Personal     | 6     | Letters, plans, personal documents                |
| Reports      | 10    | Status reports, metrics dashboards, summaries     |

Filter by domain or search by name, copy the source, and merge your data with [`parseAndMerge`](../reference/templates) or the [editor's template mode](./editor#template-mode).

The 8 built-in [themes](./themes) are pre-installed everywhere — no Hub download needed.

:::note Planned
`dotit hub` CLI subcommands (login / publish / pull) and community publishing are **planned** — today the Hub is browse-and-copy via the web app.
:::
